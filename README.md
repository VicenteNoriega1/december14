# ChatGPT Time App

A minimal ChatGPT App using the OpenAI Apps SDK that responds with the current time.

## Features

- Implements a `get_current_time` tool that returns the current time in ISO 8601 format
- Uses the system clock of the runtime
- Built with the official OpenAI Apps SDK

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Tool

The app provides a `get_current_time` tool that:
- Returns the current time in ISO 8601 format
- Uses the system clock to get accurate time information
